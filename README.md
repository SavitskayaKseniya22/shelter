# [Shelter](https://shelter-lemon.vercel.app/)
Pet shelter landing page.

## Features
- Animals carousel
- Modals
- Burger menu
- Pagination
- Mobile layout

## Tech Stack
- React
- Next
- Typescript
- Sass
- Prettier
- Eslint
- Normalize.css
- Heroicons
- Vercel

## How to use
1. Clone the repo
``` bash
git clone https://github.com/SavitskayaKseniya22/shelter.git
```

2. Install dependencies
``` bash
npm install
```

3. Start the dev server
``` bash
npm run dev
```
